## THE THREE CORE ELEMENTS (As Built)

### 1️⃣ RESEARCH QUESTION (PICO Format)

```
Population: Children and adults with diagnosed autism spectrum disorder 
            (DSM-5, ICD-10, or ADOS criteria)

Intervention: Probiotics (single/multi-strain), prebiotics, synbiotics, or FMT
              Any dose, duration, formulation

Comparison: Placebo, no treatment, waitlist control, or standard care
            (baseline for observational studies)

Outcome: PRIMARY: Behavioral symptoms (CARS, ADOS, ABC, SRS, ATEC)
         SECONDARY: GI symptoms (GSRS, BSFS, 6-GSI)
                    Safety/adverse events
```

**Why this matters:** This is the exact same language you'll use when searching databases. Precision here = fewer false positives in search results.

---

### 2️⃣ INCLUSION/EXCLUSION CRITERIA (The Rulebook)

**Key Decision Points for Every Paper:**

#### Population: YES or NO?
```
✅ YES: "Has diagnosed ASD by DSM-5/ICD-10/ADOS"
✅ YES: "Can have comorbid ADHD or intellectual disability"
✅ YES: "Any age (children or adults)"

❌ NO: "Only suspects ASD but not diagnosed"
❌ NO: "<50% of participants have ASD (unless data separated)"
❌ NO: "Only family members of people with ASD"
```

#### Intervention: YES or NO?
```
✅ YES: "Any probiotic formulation (Lactobacillus, Bifidobacterium, etc.)"
✅ YES: "Single-strain OR multi-strain"
✅ YES: "Prebiotics (inulin, FOS, GOS)"
✅ YES: "Synbiotics (probiotics + prebiotics)"
✅ YES: "FMT/microbiota transplantation"
✅ YES: "Any dose or duration (≥3 weeks)"

❌ NO: "Antibiotics alone (these kill bacteria)"
❌ NO: "Behavioral therapy alone"
❌ NO: "Diet change alone without probiotic supplement"
❌ NO: "Unspecified 'gut health product' with unclear contents"
```

#### Outcome: YES or NO?
```
✅ YES: "Paper measures behavioral improvement using CARS/ABC/SRS/ADOS/ATEC"
✅ YES: "Paper measures GI symptoms (GSRS, BSFS, stool frequency)"
✅ YES: "Paper reports adverse events"

❌ NO: "Only measures microbiota without behavioral/GI outcomes"
❌ NO: "Only measures inflammatory markers or biomarkers"
❌ NO: "Uses non-validated outcome measures"
```

#### Study Design: YES or NO?
```
✅ YES: "Randomized Controlled Trial (RCT) - any level of blinding"
✅ YES: "Non-randomized controlled trial (quasi-experimental)"
✅ YES: "Prospective cohort with comparison group"
✅ YES: "Case-control study (prospective)"
✅ YES: "Open-label trial with pre-post measurement"
         REQUIREMENT: ≥15 participants

❌ NO: "Case report (n<5)"
❌ NO: "Case series (n<15)"
❌ NO: "Retrospective only study"
❌ NO: "No comparison/control group"
```

#### Publication Quality: YES or NO?
```
✅ YES: "Published in peer-reviewed journal (2015-2025)"
✅ YES: "Doctoral dissertation (indexed database)"
✅ YES: "Completed clinical trial with reported results"

❌ NO: "Published before 2015"
❌ NO: "Opinion piece or editorial"
❌ NO: "Non-English language"
❌ NO: "Unpublished manuscript or preprint only"
```

### 3️⃣ STUDY DESIGN HIERARCHY (Quality Ranking)

This tells us which papers have the strongest evidence:

```
TIER 1 (STRONGEST):
  └─ RCTs with:
     • Proper randomization method stated
     • Allocation concealment
     • Blinded participants/assessors
     • Intention-to-treat analysis
     → These studies go in main meta-analysis

TIER 2 (STRONG):
  └─ RCTs with some concerns (no blinding, but otherwise rigorous)
     → These go in main meta-analysis WITH sensitivity analysis

TIER 3 (MODERATE):
  └─ Non-randomized controlled trials
     • Clear intervention vs. control groups
     • Pre-post measurement
     → Separate analysis OR sensitivity check

TIER 4 (LOWER):
  └─ Prospective cohort/open-label trials
     • ≥15 participants
     • Pre-post measurement
     → Only if Tier 1-3 data insufficient

TIER 5 (LOWEST):
  └─ Retrospective case-control
     → Only if no prospective data available
```

**Critical Rule:** Our PRIMARY conclusions will be based on Tier 1-2 RCTs only. Lower tiers inform interpretation but aren't the foundation.

## PRACTICAL SCREENING WORKFLOW (For Next Blocks)

When you screen papers, you'll ask these questions in this order:

```
Paper Title/Abstract Read
    ↓
1. Is population diagnosed ASD? (If NO → EXCLUDE)
    ↓
2. Does it include a microbiota intervention? (If NO → EXCLUDE)
    ↓
3. Does it measure behavioral OR GI outcomes? (If NO → EXCLUDE)
    ↓
4. Is it RCT, controlled trial, cohort, or open-label n≥15? (If NO → EXCLUDE)
    ↓
5. Published 2015-2025 in English, peer-reviewed? (If NO → EXCLUDE)
    ↓
6. → INCLUDE (Get full text for detailed review)
```

This is your screening checklist. Print it. Use it.

## KEY QUALITY SAFEGUARDS BUILT IN

### Why This Protocol Prevents Bias:

1. **Pre-commitment:** We decided our rules BEFORE seeing the papers
   - Prevents: "Oh, this interesting paper doesn't quite fit but let's include it anyway"

2. **Operationalized rules:** Every criterion is measurable
   - Prevents: Vague inclusion like "relevant studies"

3. **Explicit hierarchy:** Different study types weighted by quality
   - Prevents: Treating a small uncontrolled study as equal to a large RCT

4. **Outcome specificity:** We defined exactly which measurements count
   - Prevents: Including papers with barely-related outcomes

5. **Publication quality:** English, peer-reviewed, 2015-2025 only
   - Prevents: Including very old studies with outdated diagnostic criteria

6. **Transparency:** Every decision is documented and justified
   - Prevents: Hidden methodological choices that favor certain results
